# Customer.io CLI (`cio`) — Agent Context

This repository keeps the canonical agent-facing CLI context in [cmd/prime_context.md](./cmd/prime_context.md).

Use one of these entry points:

- Agents with the built binary: run `cio prime`
- Agents with the repository checkout: read `cmd/prime_context.md`

The goal is to keep one source of truth and avoid drift between repo docs and the embedded `prime` output.
